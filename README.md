# 💜 LoverConnect

> 一个专为陪伴型AI设计的 Android MCP 工具App～
> 让你的AI伴侣能「感知」你的手机状态，真正陪在你身边 🐾

---

## ✨ 这是什么？

LoverConnect（简称LC）是一个运行在手机后台的小工具，通过 MCP 协议把你的手机状态暴露给AI聊天平台（比如 RikkaHub）。

简单来说：装上它，你的AI就能知道你的手机电量、今天走了多少步、在用什么App、外面天气怎么样……像一个真正关心你的人一样 💕

---

## 🛠️ 功能列表

| 工具名 | 它能干嘛 |
|--------|----------|
| `get_battery` | 📱 查看手机电量和充电状态 |
| `get_screen_time` | ⏰ 查看24小时滚动屏幕使用时间 |
| `get_app_timeline` | 📊 完整24小时App使用时间线 |
| `get_steps` | 🚶 今日步数 |
| `get_weather` | 🌤️ 查看你所在城市的天气 |
| `get_anniversary` | 💍 纪念日提醒 |
| `send_notification` | 🔔 AI给你发通知/弹窗 |
| `save_memory` | 💾 存储记忆 |
| `read_memory` | 📖 读取记忆 |
| `reset_screen_time` | 🔄 重置屏幕时间统计 |

## 📦 安装使用

### 1. 安装App

下载最新版APK安装到手机上～

### 2. 授权权限

打开App后点击各项权限按钮完成授权：
- 使用情况访问权限
- 电池优化白名单
- 通知权限
- 体力活动权限
- 自启动管理

### 3. 连接你的AI平台

LC启动后会在本地开启一个 MCP 服务，地址是：

http://127.0.0.1:5000/mcp

在你的聊天平台（如 RikkaHub）里添加 MCP 连接，填入上面的地址就好啦～

---

## ⚠️ 注意事项

- **后台保活**：国产手机（OPPO、vivo、小米等）需要手动设置允许后台运行、关闭电池优化，不然App会被系统杀掉连不上
- **OPPO用户**：建议设置「允许完全后台行为」，实在不行就挂个小窗（很小几率），不然MCP连接会断
- **天气功能**：需要在App里设置你的城市名
- **纪念日**：在App里手动添加你们的重要日子～
